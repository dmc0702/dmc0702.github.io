中国剩余定理（CRT）

引入：一个古老的数学问题

在古代中国的《孙子算经》中，有一个著名的“物不知数”问题：

今有物不知其数，三三数之剩二，五五数之剩三，七七数之剩二，问物几何？

翻译成现代数学语言：
有一个整数 xxx，它除以 3 余 2，除以 5 余 3，除以 7 余 2。求这个数最小是多少？

这就是一个同余方程组：

{x≡2(mod3)x≡3(mod5)x≡2(mod7) \begin{cases} x \equiv 2 \pmod{3} \\ x \equiv 3 \pmod{5} \\ x \equiv 2 \pmod{7} \end{cases} ⎩⎨⎧​x≡2(mod3)x≡3(mod5)x≡2(mod7)​

如何求解？聪明的古人发现了一个通用解法，后来被称作中国剩余定理。

回顾：同余与模运算

在正式学习定理之前，我们需要回顾几个基本概念：

  • 同余：如果两个整数 aaa 和 bbb 除以 mmm 的余数相同，就说 aaa 和 bbb 模 mmm 同余，记作 a≡b(modm)a \equiv b \pmod{m}a≡b(modm)。
  • 模数：除数 mmm 叫做模数。
  • 互质：两个数的最大公约数为 1，称它们互质。例如 3 和 5 互质，3 和 9 不互质（gcd=3）。
  • 模逆元：如果 a×b≡1(modm)a \times b \equiv 1 \pmod{m}a×b≡1(modm)，那么 bbb 叫做 aaa 模 mmm 的逆元，记作 a−1(modm)a^{-1} \pmod{m}a−1(modm)。逆元存在的条件是 aaa 和 mmm 互质。

重要引理：

  • 如果 a≡b(modm)a \equiv b \pmod{m}a≡b(modm) 且 c≡d(modm)c \equiv d \pmod{m}c≡d(modm)，则 a+c≡b+d(modm)a+c \equiv b+d \pmod{m}a+c≡b+d(modm)，a×c≡b×d(modm)a \times c \equiv b \times d \pmod{m}a×c≡b×d(modm)。
  • 若 a×b≡a×c(modm)a \times b \equiv a \times c \pmod{m}a×b≡a×c(modm) 且 aaa 与 mmm 互质，则 b≡c(modm)b \equiv c \pmod{m}b≡c(modm)（消去律）。

中国剩余定理

标准形式：
设 m1,m2,…,mkm_1, m_2, \dots, m_km1​,m2​,…,mk​ 是两两互质的正整数（即任意两个的最大公约数都是 1），
给定任意整数 a1,a2,…,aka_1, a_2, \dots, a_ka1​,a2​,…,ak​，则同余方程组

{x≡a1(modm1)x≡a2(modm2)⋮x≡ak(modmk) \begin{cases} x \equiv a_1 \pmod{m_1} \\ x \equiv a_2 \pmod{m_2} \\ \quad \vdots \\ x \equiv a_k \pmod{m_k} \end{cases} ⎩⎨⎧​x≡a1​(modm1​)x≡a2​(modm2​)⋮x≡ak​(modmk​)​

一定有解，并且在模 M=m1m2⋯mkM = m_1 m_2 \cdots m_kM=m1​m2​⋯mk​ 下有唯一解。

这个解可以表示为：

x≡∑i=1kai⋅Mi⋅invi(modM) x \equiv \sum_{i=1}^k a_i \cdot M_i \cdot \text{inv}_i \pmod{M} x≡i=1∑k​ai​⋅Mi​⋅invi​(modM)

其中：

  • M=m1m2⋯mkM = m_1 m_2 \cdots m_kM=m1​m2​⋯mk​
  • Mi=MmiM_i = \frac{M}{m_i}Mi​=mi​M​（即去掉 mim_imi​ 后的乘积）
  • invi\text{inv}_iinvi​ 是 MiM_iMi​ 模 mim_imi​ 的逆元，即满足 Mi⋅invi≡1(modmi)M_i \cdot \text{inv}_i \equiv 1 \pmod{m_i}Mi​⋅invi​≡1(modmi​)

证明（构造性证明）

我们通过构造一个解来证明定理。

构造思路：
我们希望找到一个 xxx，使得对于每个 iii，x≡ai(modmi)x \equiv a_i \pmod{m_i}x≡ai​(modmi​)。
注意到 Mi=M/miM_i = M / m_iMi​=M/mi​ 与 mim_imi​ 互质（因为 mim_imi​ 与其他所有模数互质，所以 MiM_iMi​ 不含因子 mim_imi​），因此 MiM_iMi​ 在模 mim_imi​ 下有逆元。
令 ti=Mi⋅invit_i = M_i \cdot \text{inv}_iti​=Mi​⋅invi​，则 ti≡1(modmi)t_i \equiv 1 \pmod{m_i}ti​≡1(modmi​)，且对于 j≠ij \neq ij=i，ti≡0(modmj)t_i \equiv 0 \pmod{m_j}ti​≡0(modmj​)（因为 MiM_iMi​ 包含 mjm_jmj​ 因子）。
那么 x=a1t1+a2t2+⋯+aktkx = a_1 t_1 + a_2 t_2 + \cdots + a_k t_kx=a1​t1​+a2​t2​+⋯+ak​tk​ 就满足：

  • 对 mim_imi​ 取模时，只有 aitia_i t_iai​ti​ 项贡献 ai⋅1a_i \cdot 1ai​⋅1，其他项都是 0，所以 x≡ai(modmi)x \equiv a_i \pmod{m_i}x≡ai​(modmi​)。

唯一性：
假设有两个解 xxx 和 yyy 都满足方程组，那么 x−y≡0(modmi)x - y \equiv 0 \pmod{m_i}x−y≡0(modmi​) 对所有 iii 成立，即每个 mim_imi​ 整除 x−yx-yx−y。由于模数两两互质，它们的乘积 MMM 也整除 x−yx-yx−y，所以 x≡y(modM)x \equiv y \pmod{M}x≡y(modM)。因此在模 MMM 下解唯一。

回到开头的题目：

模数两两互质，所以可用 CRT。

  • m1=3,a1=2m_1 = 3, a_1 = 2m1​=3,a1​=2

代码实现

实现CRT需要两个关键函数：

  • 扩展欧几里得算法：求 ax+by=gcd⁡(a,b)ax + by = \gcd(a, b)ax+by=gcd(a,b) 的一组解，用于求模逆元。(为什么不用欧拉定理求逆元？为什么不用线性递推求逆元？)
  • 中国剩余定理主函数：接收模数数组和余数数组，返回解。

模数互质情况

// 扩展欧几里得算法，返回 gcd(a,b)，并求出 x, y 使得 a*x + b*y = gcd
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    long long d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}

// 求 a 模 m 的逆元，要求 a 与 m 互质
long long inv(long long a, long long m) {
    long long x, y;
    exgcd(a % m, m, x, y);
    return (x % m + m) % m;  // 保证结果非负
}

long long a[MAXN], m[MAXN];
int n;
// n 个方程，CRT 求解 x ≡ a[i] (mod m[i])，其中 m[i] 两两互质
// 返回模 M 下的唯一解
long long crt() {
    long long M = 1;
    for (int i = 1; i <= n; i++) M *= m[i];

    long long ans = 0;
    for (int i = 1; i <= n; i++) {
        long long Mi = M / m[i];
        long long invMi = inv(Mi, m[i]);  // Mi 模 m[i] 的逆元
        ans = (ans + a[i] * Mi % M * invMi % M) % M;
    }
    return ans;
}

代码分析：时间复杂度 O(n∗log⁡m)O(n*\log m)O(n∗logm)，空间复杂度 O(n)O(n)O(n)。
从代码中可以看出，因为要直接算出 MMM，nnn 和 mim_imi​ 都不会太大。
这个代码模板有没有问题？

答： 如果 MMM 在 long long 范围，那么 exgcd 的 x,y 和 ans 的计算需要__int128
ans = (ans + (__int128) a[i] * Mi % M * invMi % M) % M;

模数不互质的中国剩余定理（扩展CRT）

如果模数不是两两互质的，方程组可能无解，也可能有解，不能直接套用标准CRT公式，而需要采用逐步合并方程的方法。这种方法称为扩展中国剩余定理（ExCRT）。

一般情况下的同余方程组：
x≡ai(modmi),i=1,…,nx \equiv a_i \pmod{m_i}, \quad i=1,\dots,nx≡ai​(modmi​),i=1,…,n
（不要求 mim_imi​ 互质）

有解的充要条件：对于任意 i,ji, ji,j，有 ai≡aj(modgcd⁡(mi,mj))a_i \equiv a_j \pmod{\gcd(m_i, m_j)}ai​≡aj​(modgcd(mi​,mj​))。

求解方法：可以通过合并方程的方式逐步求解。例如，先解前两个方程，得到通解形式 x≡x0(modlcm(m1,m2))x \equiv x_0 \pmod{\mathrm{lcm}(m_1, m_2)}x≡x0​(modlcm(m1​,m2​))，然后与第三个方程合并，依此类推。

算法思路

假设我们已经合并了前 k−1k-1k−1 个方程，得到解的形式为：
x≡X(modM)x \equiv X \pmod{M}x≡X(modM)
其中 M=lcm(m1,m2,…,mk−1)M = \mathrm{lcm}(m_1, m_2, \dots, m_{k-1})M=lcm(m1​,m2​,…,mk−1​)。

现在要合并第 kkk 个方程：
x≡ak(modmk)x \equiv a_k \pmod{m_k}x≡ak​(modmk​)

即我们需要找到 ttt 使得：
X+M⋅t≡ak(modmk)X + M \cdot t \equiv a_k \pmod{m_k}X+M⋅t≡ak​(modmk​)
整理得：
M⋅t≡ak−X(modmk)M \cdot t \equiv a_k - X \pmod{m_k}M⋅t≡ak​−X(modmk​)

这是一个线性同余方程。令 d=gcd⁡(M,mk)d = \gcd(M, m_k)d=gcd(M,mk​)。方程有解当且仅当 d∣(ak−X)d \mid (a_k - X)d∣(ak​−X)。

若有解，两边除以 ddd：
Md⋅t≡ak−Xd(modmkd)\frac{M}{d} \cdot t \equiv \frac{a_k - X}{d} \pmod{\frac{m_k}{d}}dM​⋅t≡dak​−X​(moddmk​​)

此时 Md\frac{M}{d}dM​ 与 mkd\frac{m_k}{d}dmk​​ 互质，可求逆元得到：
t≡(ak−Xd)⋅(Md)−1(modmkd)t \equiv \left( \frac{a_k - X}{d} \right) \cdot \left( \frac{M}{d} \right)^{-1} \pmod{\frac{m_k}{d}}t≡(dak​−X​)⋅(dM​)−1(moddmk​​)

设 t0t_0t0​ 为上述模意义下的解，则通解为 t=t0+mkd⋅kt = t_0 + \frac{m_k}{d} \cdot kt=t0​+dmk​​⋅k。代入 x=X+M⋅tx = X + M \cdot tx=X+M⋅t，得到新解：
x≡X+M⋅t0+M⋅mkd⋅k≡X+M⋅t0(modlcm(M,mk))x \equiv X + M \cdot t_0 + M \cdot \frac{m_k}{d} \cdot k \equiv X + M \cdot t_0 \pmod{\mathrm{lcm}(M, m_k)}x≡X+M⋅t0​+M⋅dmk​​⋅k≡X+M⋅t0​(modlcm(M,mk​))

其中 lcm(M,mk)=M⋅mkd\mathrm{lcm}(M, m_k) = \frac{M \cdot m_k}{d}lcm(M,mk​)=dM⋅mk​​。

重复这个过程，直到所有方程合并完毕。最终得到的 XXX 就是原方程组的一个解，最终的模数为最终合并后的模数 M=lcm(m1,...,mn)M = \mathrm{lcm}(m_1,...,m_n)M=lcm(m1​,...,mn​)。

扩展CRT代码

#define ll long long
// 扩展欧几里得算法，返回 gcd(a,b)，并求出 x, y 使得 a*x + b*y = gcd
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    ll d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}

// 求 a 模 m 的逆元，要求 a 与 m 互质
ll inv(ll a, ll m) {
    ll x, y;
    exgcd(a % m, m, x, y);
    return (x % m + m) % m;
}

// 合并两个同余方程：x ≡ a1 (mod m1), x ≡ a2 (mod m2)
// 返回合并后的方程 (new_a, new_m)，若合并失败（无解）则返回 ( -1, -1 )
void merge(ll a1, ll m1, ll a2, ll m2, ll &new_a, ll &new_m) {
    ll d = __gcd(m1, m2);  // d = gcd(m1, m2)
    ll c = a2 - a1;
    if (c % d != 0) { // 无解
        new_a = -1, new_m = -1;
        return;
    }

    // 解方程 m1 * t ≡ c (mod m2)  ->  (m1/d) * t ≡ c/d (mod m2/d)
    m1 /= d, m2 /= d, c /= d;

    // 求 m1/d 关于模 m2/d 的逆元
    ll inv_m1 = inv(m1, m2);
    // t0 ≡ c/d * inv_m1 (mod m2/d)
    ll t0 = (c * inv_m1 % m2 + m2) % m2;

    // 新模数为 lcm(m1, m2) = m1/d * m2/d * d
    new_m = m1 * m2 * d;  // 注意乘法可能溢出，但题目数据一般保证在 long long 范围内
    new_a = ((a1 + m1 * d * t0) % new_m + new_m) % new_m;
}
ll a[MAXN], m[MAXN];
int n;
// 扩展中国剩余定理：求解 x ≡ a[i] (mod m[i])，返回 (解 now_a, 模数 now_m)，若无解返回 (-1, -1)
void excrt(ll &now_a, ll &now_m) {
    ll now_a = (a[1] % m[1] + m[1]) % m[1], now_m = m[1];
    for (int i = 2; i <= n; i++) {
        merge(now_a, now_m, a[i], m[i], now_a, now_m);
        if (now_m == -1) { // 无解
            return;
        }
    }
}

练习题

例题：
解方程组
x≡2(mod4),x≡4(mod6) x \equiv 2 \pmod{4},\quad x \equiv 4 \pmod{6} x≡2(mod4),x≡4(mod6)

分析：
先判断：gcd(4,6)=2，看模 2 意义下 x 的值会不会产生矛盾，2 ≡ 4 mod 2，相等，有解。
设 x=2+4tx = 2 + 4tx=2+4t，代入第二个：2+4t≡4(mod6)2+4t \equiv 4 \pmod{6}2+4t≡4(mod6) → 4t≡2(mod6)4t \equiv 2 \pmod{6}4t≡2(mod6)。除以 2：2t≡1(mod3)2t \equiv 1 \pmod{3}2t≡1(mod3)，2 在模 3 下的逆元是 2，所以 t≡2(mod3)t \equiv 2 \pmod{3}t≡2(mod3)，即 t=2+3kt = 2 + 3kt=2+3k。代回：x=2+4(2+3k)=10+12kx = 2 + 4(2+3k) = 10 + 12kx=2+4(2+3k)=10+12k，所以 x≡10(mod12)x \equiv 10 \pmod{12}x≡10(mod12)。

应用

  1. 解同余方程组：直接使用CRT。给定一些同余方程，求最小正整数解。有时要判断解的存在性，或与扩展欧几里得算法结合使用。
  2. 大整数取模：当模数很大但可以分解为若干互质的小模数时，可以在每个小模数下计算，再用CRT合并结果，避免大数运算。
  3. 组合计数：某些计数问题需要模一个大合数，而大合数可以分解为互质因子的乘积，分别计算后用CRT合并。
  4. RSA密码：涉及模幂运算，有时用CRT加速解密。

小结

  • 中国剩余定理解决了一类特殊的同余方程组问题，核心是构造解。
  • 模数两两互质时，直接用解的公式求解；若不互质，需先判断有解条件，再用合并方程的方式求解。
  • 实现时主要用到扩展欧几里得求逆元。
  • CRT 常与其他数论知识结合，需要熟练掌握算法模板。
